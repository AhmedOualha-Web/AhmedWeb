Network and System Administration

Number Course Name Course Description
1 Network Foundations Learned the layers of the Open Systems Interconnection Model and enable them to describe the features and functions of network devices (protocols, topologies, networking standards)
2 Enterprise Mail Systems Learned planning, installation, configuration, and support of enterprise mail systems
3 Network Implementation Learned various types of devices, protocols and technologies associated with connecting networks, both LANs and WANs.
4 Wireless Networks Learned a broad survey of wireless communications….
5 Network Operating System Administration Learned the skills to install, configure, optimize, troubleshoot, and support a network server with day-to-day administration.
6 Emerging Trends in IT Infrastructure Learned new trends in IT infrastructure and evolving techniques, technologies used in the design and maintenance of the IT infrastructure.
7 Web Server Management Learned web server management using Microsoft Internet Information Server and Apache Web Server (installation procedures, configuration of virtual hosts and redirects, web server hardening and E-Commerce security considerations and web server administration)
8 Advanced Networking 1 Learned IP-based communications with the concept of growing an IP network (VLAN and Inter-VLAN routing, LAN redundancy, link aggregation, and advanced routing concepts)
9 Enterprise Technology 1 Learned skills to install, configure and manage the core services of a corporate server (TCP/IP configuration; routing; name service configuration; and user, computer, and group strategy deployment)
10 Enterprise Technology 2 Learned skills to implement, manage, and maintain remote access, configure web servers, and secure internet access and data transmissions.
11 Advanced Networking 2 Learned Internetworking: Routing (distant vectors routing protocols and linked state routing protocols), switching (spanning tree protocols, virtual LANs, and VLAN trunking protocols), and WAN technologies
12 Computer Network Protocols Learned to install, operate, configure, and verify basic IPv4 and IPv6 networks (configuring network components such as switches, routers, and wireless LAN controllers; managing network devices; and identifying basic security threats)
